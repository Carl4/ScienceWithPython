Links

  • Big Machine Learned Astronomical Classification Catalog Huge catalog of more than 50,000 probabalistically classified variable stars.

    • Catalog built using rpy2, scipy, matplotlib
    • Paper: Richards et al. (arxiv/1204.4180)
    • Website built using Google App Engine, jquery, and Google Fusion Tables

  • dotastro.org: astronomical light curve repository ay UC Berkeley.

    • built using Django

  • Astrometry

    • PyMPChecker Parallelized Python Minor Planet checker webservice (Chris Klein)
      • Django, pyephem
    • Astrometry.net Astrometry webservice for images (David Hogg, NYU et al.)
    • pyBAST Bayesian Astrometry in Python github project (Berian James; JSB)

  • PAIRITEL Peters Automated Infrared Imaging Telescope

    • robotized 1.3 meter telescope in Mt Hopkins, Arizona.
    • project started in 2003 (Python 2.2)
    • built using Python (pyserial, pyparallel)
    • pygcnsock: Gamma-ray Burst Trigger Software (Old Project!)

  • Center for Time-Domain Informatics at UC Berkeley.

  • Python Bootcamp at UC Berkeley

  • Quasar Fitting with scipy See also the associated paper in the Astronomical Journal by Butler and Bloom.

  • Palomar Transient Factory (PTF)

  • Large Synoptic Survey Telescope
